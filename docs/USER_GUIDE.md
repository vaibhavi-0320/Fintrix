# User Guide

## Quick Start

1. Launch the app and connect a wallet.
2. Open **Marketplace** to browse invoice opportunities.
3. Use **Fintrix AI** for risk checks before funding decisions.
4. Go to **Simulation** to create a new invoice simulation.
5. Track events in **Activity** and overview in **Dashboard**.

## Marketplace

- Filter by status, risk, APY, and duration.
- Review risk badge, yield, and funding progress on each card.
- Use **Fund Now** on active opportunities.

## Simulation

- Upload invoice files (PDF/image) or enter fields manually.
- Submit to create a simulation and list it in marketplace.
- Review execution cards and recent events.

## AI Assistant

- Click the AI orb to open chat.
- Use quick prompts for pre-funding checklists.
- Review Risk Detector guidance before decisions.

## Troubleshooting

- If wallet connection fails, retry from the connect action.
- If backend is unavailable, app falls back to demo-safe data.
- Reload application if transient network/parsing errors occur.
